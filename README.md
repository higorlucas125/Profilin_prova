# Profilin_prova

## 📚 Sobre este repositório

Este repositório é destinado **exclusivamente ao armazenamento de provas** realizadas no Profilin.

## 📁 Estrutura de cada prova

Para cada prova registrada, serão incluídos dois arquivos:

| Arquivo | Descrição |
|---------|-----------|
| `prova.pdf` | PDF com o enunciado original da prova |
| `respostas.md` | Arquivo Markdown com as respostas desenvolvidas |

## 🎯 Objetivo

Este repositório serve como **controle de autoria pessoal**, garantindo um histórico organizado e rastreável de todas as provas e suas respectivas respostas.

---

> **Nota:** O conteúdo deste repositório é de uso pessoal e visa apenas o registro e controle das próprias atividades avaliativas.
